## 🧵 Introducción / Introduction

Este proyecto explora el caso de **Arcor**, una de las marcas más icónicas de Latinoamérica, desde una perspectiva estratégica y técnica, combinando análisis de datos, experiencia de usuario (UX) y marketing digital moderno.

*This project explores the case of **Arcor**, one of Latin America's most iconic brands, from a strategic and technical perspective, combining data analysis, user experience (UX), and modern digital marketing.*

### 🎯 Alcance del Proyecto / Project Scope

- ✅ Análisis UX/UI del sitio oficial de Arcor
- ✅ Benchmarking competitivo con Nestlé y Mondelez
- ✅ Visualizaciones interactivas de datos de consumo
- ✅ Propuestas de sostenibilidad y packaging ecológico
- ✅ Auditoría de usabilidad con métricas Lighthouse
- ✅ Estrategias de marketing digital data-driven

## 🚀 Instalación y Despliegue / Installation & Deployment

### Despliegue Rápido en GitHub Pages / Quick GitHub Pages Deployment

1. **Fork el repositorio** o clónalo localmente
```bash
git clone https://github.com/tu-usuario/Arcor-estrategia-de-Marketing-UX-basada-en-datos.git
```

2. **Configurar GitHub Pages**
   - Ve a `Settings > Pages` en tu repositorio
   - Selecciona `Source: Deploy from a branch`
   - Elige `main` branch y `/ (root)` folder
   - Guarda la configuración

3. **Acceder al sitio**
   - Tu sitio estará disponible en: `https://tu-usuario.github.io/repo-name`
   - El despliegue tarda 1-2 minutos

### Desarrollo Local / Local Development

```bash
# Clonar repositorio
git clone https://github.com/tu-usuario/repo-name.git

# Navegar al directorio
cd Arcor-estrategia-de-Marketing-UX-basada-en-datos

# Servir localmente (Python)
python -m http.server 8000

# O con Node.js
npx http-server

# Acceder en: http://localhost:8000
```
